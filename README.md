\# Augmentiverse Cat AR — MindAR



This project anchors `happy.glb` to a printed cat image using MindAR image tracking.



\## Files



\- `index.html`

\- `happy.glb`

\- `cat-print-target.png`

\- `targets.mind`

\- `augmentiverse-manifest.json`



\## Step 1: Prepare the printed target



Rename your cat image to:



```text

cat-print-target.png



Compile the cat image with the MindAR image target compiler:



https://hiukim.github.io/mind-ar-js-doc/tools/compile/
